# Pennywise – Smart Savings Calculator

Pennywise is a simple, and smart savings calculator that helps users estimate how long it will take to reach their financial goals.
Built with HTML, CSS, and JavaScript, it combines a clean design with an intuitive user experience.

# Features

- **Smart Goal Calculator** – Enter your target amount, current savings, and monthly savings to see how long it’ll take to reach your goal.
- **Clean, Soothing UI** – Designed with a calm, modern color palette to make financial planning.
- **Instant Feedback** – Get results dynamically — no reloads required.

# Tech Stack

- HTML5 – Structure and layout
- CSS3 – Styling and color palette
- JavaScript – Core logic and user interaction
